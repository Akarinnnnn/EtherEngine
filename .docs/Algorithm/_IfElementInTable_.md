# IfElementInTable

> 判断指定元素是否位于表值中

```lua

flag = IfElementInTable(element, table, type)

```

## 参数：

+ element [number, string]：指定元素，类型为number或string
+ table [table]：表，值类型与被判断元素类型相同
+ type [Macro number]：元素类型，可选值如下：
    
    + ELEMENTTYPE_NUM：number类型元素
    + ELEMENTTYPE_STR：string类型元素

## 返回值：

+ flag [boolean]：是否存在
