# CreateTextImageSolid

> 使用Solid模式创建文本图像

```lua

image = CreateTextImageSolid(font, text, color)

```

## 参数：

+ font [userdata-FONT]：字体数据
+ text [string]：文本内容
+ color [table]：用以描述文本颜色的RGBA表，各分量取值范围均为0-255

## 返回值：

+ image [userdata-IMAGE]：成功则返回图像数据，失败则返回nil
