# ShowMessageBox

> 显示模态的提示信息窗口

```lua

ShowMessageBox(title, content, type)

```

## 参数：

+ title [string]：提示窗口标题
+ content [string]：提示信息内容
+ type [Macro number]：窗口类型，可选值为：

    + MSGBOX_ERROR：错误
    + MSGBOX_WARNING：警告
    + MSGBOX_INFO：提示

## 返回值：

无
