# IfCirclesOverlap

> 判断两圆是否相交（包含圆形边界）

```lua

flag = IfCirclesOverlap(center1, radius1, center2, radius2)

```

## 参数：

+ center1 [table]：圆1圆心坐标
+ radius1 [number]：圆1半径
+ center2 [table]：圆2圆心坐标
+ radius2 [number]：圆2半径

## 返回值：

+ flag [boolean]：是否相交
